# RPG - Combate Melhorado

Melhorias:
- Inimigos e chefão agora atacam olhando corretamente para o player.
- Corrigida direção dos sprites.
- Hitbox do player melhorada.
- Hitbox dos inimigos melhorada.
- Área de ataque mais precisa.
- Combate mais justo e responsivo.
- Ataques não acertam mais de costas.

Controles:
WASD = mover
ESPAÇO = atacar
R = reiniciar
